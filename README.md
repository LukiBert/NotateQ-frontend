# NotateQ Frontend Repo

## Project Setup

Download or clone this repo:

```sh
git clone git@github.com:LukiBert/NotateQ-frontend.git
```

TProject is build with **yarn**, so run the following commands:

```sh
yarn
yarn dev
yarn build
```
